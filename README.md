# 智能衣橱 - AI穿搭助手

一个基于 Next.js 的智能穿搭推荐应用，帮助你根据天气、场合和个人衣橱找到最佳穿搭方案！

## ✨ 功能特性

### 1. 衣橱管理
- 📷 **拍照/上传**：轻松添加新衣物
- 🏷️ **分类管理**：上衣、裤子、外套、鞋包、配饰等
- 📦 **标签系统**：按季节、风格等智能分类

### 2. 天气驱动
- ☀️ **实时天气**：根据当日天气推荐合适衣物
- 🌧️ **天气选择**：晴天、雨天、雪天、大风等多种模式
- 🌡️ **温度感知**：自动推荐适合温度的穿搭

### 3. 场合定制
- 👔 **上班/商务**：专业得体的办公穿搭
- ❤️ **约会**：浪漫时尚的约会造型
- 🎉 **聚会/节日**：亮眼吸睛的派对装扮
- ✈️ **旅行**：舒适实用的出行搭配

### 4. 语音交互
- 🎤 **语音指令**：通过语音直接告诉AI你的需求
- 🔍 **智能识别**：自动解析场合和天气偏好

### 5. 智能推荐
- 🤖 **AI算法**：基于天气、场合和个人衣橱的智能匹配
- ⭐ **搭配指数**：直观显示推荐搭配的合适程度
- 🔄 **换一批**：不满意？一键刷新获得新推荐

## 🚀 快速开始

### 环境要求
- Node.js 18+ 
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000)

### 构建生产版本
```bash
npm run build
npm start
```

## 🎯 使用说明

### 添加衣物
1. 点击底部「衣橱」标签
2. 点击右上角 + 按钮
3. 选择拍照或从相册上传
4. 填写衣物名称和分类
5. 保存到衣橱

### 获取推荐
1. 点击底部「推荐」标签
2. 确认或选择今日天气
3. 选择今日出行场合
4. 浏览推荐的穿搭方案
5. 点击「就穿这套」或「换一批」

### 语音指令
1. 点击右上角麦克风图标
2. 说出你的需求（如：「今天约会穿什么」）
3. 等待AI解析并自动推荐
